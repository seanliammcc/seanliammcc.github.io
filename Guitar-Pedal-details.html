Guitar Pedal Overdive

  • Category: Technical Project
  • Project date: April - August 2020,   January 2021 - Present

Project Description

Guitar Pedals are used to alter the waveforms created by electric guitars to produce a different sound. In this project, the signal was amplified past the threshold of which the amp could reconstruct AC into sound, creating Overdrive. Additionally, higher level harmonics were introduced to create distortion.
