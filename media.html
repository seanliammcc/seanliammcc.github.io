this page is the answer to the question: Have you read/seen anything good, lately?

here's a list of media that i think are worth a try.
 
the date is (roughly) the first time i watched/read/listened/finished.
 
i use tags to help you find the sort of thing you're looking for.
these are the tags i've used so far: audiobook, book, documentary, essay, fiction, film, non-fiction, novel, novella, play, podcast, poem, short story, television.
 
some tags are redundant (e.g. novel & fiction, documentary & non-fiction). this is intended to make searching easier.  
 
please let me know if you have any ideas how to make this list more useful.
 
enjoy!
